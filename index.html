⚙️ Conectar Google Sheet

Solo necesitas publicar el Apps Script como Web App y pegar la URL aquí. Sin API Keys.

Pasos (solo una vez):
  1. En tu Google Sheet → Extensiones → Apps Script
  2. Pega el código del archivo apps_script_code.js
  3. Clic en Implementar → Nueva implementación
  4. Tipo: Aplicación web
  5. Ejecutar como: Yo  ·  Acceso: Cualquier persona
  6. Autoriza y copia la URL → pégala arriba
